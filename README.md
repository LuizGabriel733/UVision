# UVision_Inovatech
A proposta do projeto UVision é criar uma pulseira inteligente de baixo custo que monitora continuamente a exposição à radiação UV. Utilizando um sensor (ML8511) para calcular a Dose Eritêmica Padrão (SED), o objetivo central é prevenir o câncer de pele e conscientizar a população, servindo como uma ferramenta de saúde preventiva acessível a todos.
